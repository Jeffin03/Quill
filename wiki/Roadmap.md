# 🗺️ Roadmap

## Phase 1: The Foundation (Core Loop) — ✅ DONE
- [x] Basic AI Chat and prose generation.
- [x] Context Card Engine (basic character tracking).
- [x] Responsive Sidebar UI.

## Phase 2: The Studio (PWA & Polish) — ✅ DONE
- [x] IndexedDB local storage.
- [x] PWA installation support.
- [x] Pacing and Tone controls.

## Phase 3: The Multiverse (Branching) — ✅ DONE
- [x] Hierarchical story trees.
- [x] Timeline forking (Fork any message).
- [x] Independent world state (Card Snapshots).
- [x] Visual Tree Navigator.

## Phase 4: Stabilization & UX — ✅ DONE
- [x] Multi-Genre support.
- [x] Model History & Live Status.
- [x] JSON Deep Repair for cards.
- [x] Local IP / non-HTTPS compatibility.

## Phase 5: Power Features (Visuals & Dynamics) — 🚀 NEXT
### Visual Identity
- **Portrait Upload**: Add images to Character Cards.
- **AI Visual Analysis**: Use Vision models to auto-populate traits from photos.
- **Scene Previews**: Dynamic visuals in the chat history.

### Relationship Mapping
- **Social Web**: A visual graph showing character dynamics (Allies, Rivals, Lovers).
- **Dynamic Links**: Map lines that evolve as story relationships change.

## Phase 6: Scale & Refinement
- **Search & Filter**: Find any node or card in massive stories.
- **Lazy Trees**: Optimize the Visual Tree for 1000+ nodes.
- **Export Pro**: EPUB/PDF professional formatting.
- **Lore Listeners**: Real-time card update suggestions while you write.
